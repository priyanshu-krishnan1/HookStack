# Code of Conduct

## Our standard

Contributors are expected to be respectful, constructive, and professional.

Examples of positive behavior:

- giving actionable technical feedback
- being respectful in disagreements
- helping newcomers understand the project
- focusing on the code and ideas, not the person

Examples of unacceptable behavior:

- harassment or abusive language
- personal attacks
- discrimination
- intentionally disruptive conduct

## Enforcement

Project maintainers may remove or edit contributions, comments, or interactions that violate this code of conduct.

## Scope

This code of conduct applies to:

- issues
- pull requests
- discussions
- code review interactions

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainer.
